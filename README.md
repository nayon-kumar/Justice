## Overview

A modern and responsive **Legal Solutions landing page** built using **HTML and Tailwind CSS**.

## 🛠 Technologies Used

- **HTML5** – Structure of the webpage
- **Tailwind CSS (CDN)** – Styling and responsive layout
- **Font Awesome** – Icons
- **Google Fonts (EB Garamond)** – Typography

## 🚀 Features

- Fully responsive landing page
- Professional legal website design
- Navigation bar with responsive menu
- Hero banner with search feature
- Legal practice areas section
- Client testimonial section
- Frequently Asked Questions (FAQ)
- Contact information and contact form
- Footer with useful links and newsletter subscription
- Smooth hover effects and transitions
